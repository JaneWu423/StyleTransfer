Image Style Transfer: Painting 2 Painting

GatedGAN

A generative adversarial network that can apply pretrained styles to a given content image. The gating function controls how much each style is applied to the content.

Style Mixer

A neural network architecture called the Semantic-aware Multi-Style Transfer Network (SMTNet), which combines the strengths of existing style transfer methods to create a more powerful and flexible model.
